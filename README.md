# Campanha de Crowdfunding "A ROSA"

Este repositório contém o website promocional fictício para a campanha de crowdfunding do curta-metragem de animação "A ROSA".

## Sobre "A ROSA"

"A ROSA" é um curta-metragem de animação poético sobre amor, perda e aceitação, contado sem palavras, através da magia da animação e da música. Nossa campanha de crowdfunding visa arrecadar os fundos necessários para dar vida a esta história emocionante e levá-la a festivais nacionais e internacionais.

## Objetivo do Site

O website serve como uma ferramenta de divulgação e informação para a campanha de crowdfunding. Ele detalha a proposta do filme, a equipe, o orçamento, as metas estendidas, as recompensas para os apoiadores e oferece conteúdo adicional como podcasts sobre o projeto. O objetivo principal é engajar potenciais apoiadores e direcioná-los para a plataforma oficial de arrecadação (Catarse).

## Estrutura do Projeto

O projeto é organizado da seguinte forma:

-   `index.html`: A página principal (home) da campanha, apresentando um resumo geral e os principais pontos.
-   `styles.css`: O arquivo CSS global para estilização da barra de navegação e outros elementos comuns.
-   `audio/`: Contém arquivos de áudio (podcasts) relacionados à história e aos bastidores da campanha.
-   `doc/`: Documentação interna do projeto, incluindo o plano estratégico da campanha.
-   `metas-estendidas/`: Página dedicada a apresentar as metas adicionais que serão alcançadas se a arrecadação exceder o objetivo inicial.
-   `pagina-de-apoio-campanha-crowdfounding/`: Uma página central para visualização e seleção das diferentes recompensas disponíveis.
-   `recompensa-*/`: Diretórios contendo as páginas de visualização de cada nível de recompensa (ex: `recompensa-semente/index.html`).

## Como Visualizar o Site

Para visualizar o site localmente, basta abrir o arquivo `index.html` em seu navegador de preferência. Não é necessário nenhum servidor web para o funcionamento básico.
